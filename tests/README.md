# Tests Directory

ไดเรกทอรีสำหรับ Unit Tests และ Integration Tests

## สิ่งที่จะมี (อนาคต)

- `test_extractor.py` - ทดสอบการแปลง PDF
- `test_transformer.py` - ทดสอบการแปลง JSON เป็น CSV
- `test_pipeline.py` - ทดสอบ Pipeline ทั้งหมด

## วิธีการรัน Tests

```bash
pytest tests/
```

หรือ

```bash
python -m pytest tests/
```
